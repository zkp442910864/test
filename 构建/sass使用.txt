安装:
	安装Ruby
		注意:安装的时候添加到path路径里,命令行才能起作用.
		https://www.ruby-lang.org/en/documentation/installation/
		Sass是用Ruby语言写的，但是两者的语法没有关系，所以学 Sass 不用学 Ruby，只是必须先安装Ruby，然后再安装Sass
		Linux和Mac已自带Ruby，不用再安装。Windows用户可以从这里下载安装包
		验证Ruby是否安装成功
			ruby -v		如果输出Ruby版本号，则表示安装成功

	安装Sass
		cmd命令行输入下面的命令
			gem install sass
		是否安装成功
			sass -v		如果输出Sass版本号，则表示安装成功

SASS文件转译
	*最好在文件目录输入cmd打开控制台.
	SASS的转译方法
		sass   style.scss   style.css
		标识   被编译文件	编译后的文件,我们需要的.

		sass   sass/style.scss    css/style.css
		可以通过路径来读文件,和存文件.

		sass --watch  style.scss
		当scss文件每次保存会自动编译成css文件,但是只能保存在当前文件夹.

		文件夹下多个scss文件都编译成css文件放大css文件夹下
		sass --watch sass/:css/

	4.混合器

	  -webkit-border-radius: $radius;
	     -moz-border-radius: $radius;
	      -ms-border-radius: $radius;

	5.继承
	.message {
	  border: 1px solid #ccc;
	  padding: 10px;
	  color: #333;
	}
	.success {
	  @extend .message;
	  border-color: green;
	}
	.error {
	  @extend .message;
	  border-color: red;
	}
	.warning {
	  @extend .message;
	  border-color: yellow;
	}
