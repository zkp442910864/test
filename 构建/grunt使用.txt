
安装grunt
	1.安装nodejs	https://nodejs.org/en/
		控制台中输入“node -v”来查看是否成功
	2.安装grunt-CLI
		控制台中输入“npm install -g grunt-cli”
		控制台中输入“grunt”来查看是否成功
	3.配置package.json
		新建个文件夹(grunt_test)->新建个package.json文件	
			内容:{
					  "name": "grunt_test",		//文件夹名
					  "version": "1.0.0",		//版本号
					  "devDependencies": {
					   		//开发的依赖
					  }
					}	
	4.安装grunt			
		命令:npm install grunt --save-dev
		“—save-dev”的意思是，在当前目录安装grunt的同时，顺便把grunt保存为这个目录的开发依赖项
		完成后再配置文件里可以看到"grunt"
	5.配置Gruntfile.js
		新建个Gruntfile.js文件(和package.json同级)
			内容:module.exports = function(grunt){
					grunt.initConfig({
						pkg:grunt.file.readJSON('package.json'),

						//插件配置信息写这,如
						uglify:{
							options:{
								stripBanners:true,
								banner:'/*! <%= pkg.name %> - <%= pkg.version %>.js <%= grunt.template.today("yyyy-mm-dd") %> */\n'
							},
							build:{
								src:'src/test.js',
								dest:'build/<%=pkg.name%>-<%=pkg.version%>.js.min.js'
							}
						}

					});
					grunt.registerTask('default',[]);
				};

				grunt.loadNpmTasks('grunt-contrib-uglify');//表示要使用到的插件

				grunt.registerTask('default',['uglify']);//进行调用
	6.完成,剩下的就是各种插件的安装使用
		http://www.gruntjs.net/plugins		插件列表			

	
批量安装插件
	在package.json文件中已列出的开发的依赖
	直接通过命令 “npm install” 安装
