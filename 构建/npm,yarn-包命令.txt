

    npm安装模块
		install === i;
		–save === -S;
		–save-dev === -D;

        大部分使用 cnpm 比较快

        【npm init】 初始化

        【npm install || npm i】 依赖安装

		【npm install xxx】利用 npm 安装xxx模块到当前命令行所在目录；
		【npm install -g xxx】利用npm安装全局模块xxx；
		本地安装时将模块写入package.json中：

		【npm install xxx】安装但不写入package.json；
		【npm install xxx –save】 安装并写入package.json的”dependencies”中；
		【npm install xxx –save-dev】安装并写入package.json的”devDependencies”中。
		npm 删除模块
		【npm uninstall xxx】删除xxx模块；
		【npm uninstall -g xxx】删除全局模块xxx；

        【npm publish/login/logout】 发布/登录/登出,一系列NPM Registry操作

        【npm cache clean --force】 清缓存

    yarn
        【yarn init】 初始化

        【yarn install】 依赖安装

        【yarn add xxx】 安装某个模块
        【yarn add xxx --dev】 安装开发时某个模块
        【yarn global add xxx】 全局安装某个模块

        【yarn remove xxx】 删除某个模块

        【yarn publish/login/logout】 发布/登录/登出,一系列NPM Registry操作

        【yarn cache clean】 清缓存
